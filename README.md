# Frisma Dental Care - Super-Premium Dental Clinic Template

Situs web super-premium, SEO-optimized, dan responsif yang dirancang khusus untuk **Klinik Gigi Frisma Dental Care - Kupang**. Template ini menonjolkan estetika modern dengan sentuhan glassmorphism, animasi halus, dan pengalaman pengguna yang intuitif.

## ✨ Fitur Utama
- **Modern Glassmorphism Design**: Antarmuka bersih dengan efek blur premium dan palet warna harmonis (Emerald & Sky).
- **SEO Optimized**: Dilengkapi dengan meta tags lengkap dan JSON-LD Structured Data (Schema.org) untuk visibilitas maksimal di Google.
- **Ultra Responsive**: Tata letak yang beradaptasi sempurna di perangkat mobile hingga desktop.
- **Fast Performance**: Dibangun dengan Vite dan React untuk kecepatan muat yang kilat.
- **WhatsApp Integration**: Tombol mengapung interaktif untuk mempermudah pasien melakukan janji temu.
- **Premium Components**: Navbar transparan, Hero section yang memukau, daftar layanan profesional, dan form kontak terintegrasi.

## 🛠️ Tech Stack
- **Frontend**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router v7](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend / Auth**: [Supabase](https://supabase.com/)

## 🚀 Persiapan & Instalasi

### 1. Clone Repository
```bash
git clone <repository-url>
cd templateSite-klinikGigi
```

### 2. Instalasi Dependencies
```bash
npm install
```

### 3. Konfigurasi Environment
Salin file `.env.example` menjadi `.env` dan isi kredensial Supabase Anda:
```
VITE_SUPABASE_URL=your-project-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```

### 4. Jalankan Development Server
```bash
npm run dev
```

### 5. Build untuk Produksi
```bash
npm run build
```

## 📁 Struktur Proyek
- `src/components/`: Komponen UI (Navbar, Hero, Services, Contact, dll).
- `src/pages/`: Halaman aplikasi (Dashboard, Editor, Pricing, dll).
- `src/core/`: Logika generator dan template builder.
- `src/index.css`: Desain sistem inti dan variabel warna global.
